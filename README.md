# 🎓 Memory Wall

A full-stack web application built for college students to share memories, moments, and experiences in one place before graduation.

## 🚀 Features

- User authentication & authorization
- Create, edit, and delete posts
- Upload images
- Responsive modern UI
- PostgreSQL database integration
- Secure backend handling
- Deployed online for public access

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python(Flask)

### Database
- PostgreSQL

### Deployment
- Replit
